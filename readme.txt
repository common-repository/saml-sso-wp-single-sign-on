WP-SSO plugin helps your users to login to WordPress site with any SAML compliant Identity provider


== Description ==
Keywoot offers a SSO Plugin


== Installation ==
= From your WordPress dashboard =
1. Navigate to Dashboard > Plugins > Add New.
2. Search for `login sso for wp`. Locate and install `WP-SSO | Single Sign On for WP | Login With SSO`.
3. Activate the plugin from your Plugins page.

= From WordPress.org =
1. Download `WP-SSO | Single Sign On for WP | Login With SSO`.
2. Unzip the downloaded file and upload the `keywoot-saml-sso` directory to your `/wp-content/plugins/` directory.
3. Activate the SSO plugin from your Plugins page.
